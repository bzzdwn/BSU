\chapter*{Заключение}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}

В дипломной работе получены следующие основные результаты:
\begin{enumerate}
	\item [1)] сформулирован и подготовлен обзор основных подходов и методов, применяемых для краткосрочного прогнозирования и наукастинга;
	\item [2)] подготовлено описание эконометрических моделей и методов, применяемых для прогнозирования прироста ВВП Республики Беларусь;
	\item [3)] разработан алгоритмический инструментарий и программное обеспечение на основе метода расширяющегося окна для краткосрочного прогнозирования по моделям векторной авторегрессии по смешанным данным;
	\item [4)] проведен сравнительный анализ моделей по смешанным данным с моделями по агрегированным данным.
\end{enumerate}

На основе полученных результатов можно сделать следующие выводы: 
\begin{itemize}
	\item модель MF-VAR по данным смешанной частоты при наилучшем подборе высокочастотных переменных способна строить более точные прогнозы по сравнению с моделью VAR по агрегированным данным в режиме краткосрочного прогнозирования; 
	\item на значения реального ВВП белорусской экономики в краткосрочной перспективе наибольшее влияние оказывают такие  макроэкономические показатели, как объем промышленного производства, объем розничного товарооборота и индекс экономических настроений СИЭН;
	\item модель MF-VAR по данным смешанной частоты способна проводить качественный наукастинг в спокойных для экономики Беларуси периодах.
\end{itemize}

Таким образом, данное исследование вносит вклад в развитие эконометрических методов анализа временных рядов по данным смешанной частоты. По результатам исследования можно утверждать, что MF-VAR модель применима к белорусской экономике для краткосрочного прогнозирования и наукастинга с целью корректировки оценки ВВП.
